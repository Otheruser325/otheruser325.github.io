Version 28.5

  • General Changes
    • The existing "One Life" game mode has been reworked to "Afterlife".
      • In "Afterlife", both players start with 1,500 lives and are slowly drained during the round.
      • The grace period of losing lives occurs every 5 seconds, so be wary and get those rounds short!
      • Also, "Salted Wounds" and "Shield" powers are unbanned in this mode (from banned in the old mode).
    • Playing certain game mode combinations with Afterlife such as "Mega Boost Afterlife" brings something special, so stay alert!
    • Updated crosspath visuals for Dart, Tack, Ice and Mortar
